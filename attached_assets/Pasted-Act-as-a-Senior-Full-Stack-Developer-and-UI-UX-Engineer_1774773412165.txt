---

- Tailwind CSS ONLY
- Node.js + Express.js

- JWT Authentication

---

🎯 OBJECTIVE:
Enable users and admins to easily find, filter, and organize content such as quizzes, study materials, and users.

🔍 SEARCH FUNCTIONALITY

📌 Where to Implement:

- User Dashboard (quizzes, study material)
- Admin Panel (users, quizzes, content)

📌 Frontend:

- Add search input field at top of lists
- Debounce input (optional for performance)
- Filter results dynamically as user types

📌 Backend (if needed):

- Support query params:
  - /api/quizzes?search=math
  - /api/users?search=john

📌 Logic:

- Match:
  - Title
  - Subject
  - Keywords

📌 Filters to Add:

- Subject (Math, Science, etc.)
- Difficulty (Easy, Medium, Hard)
- Date (recent)

For Users (Admin):

- Role (user/admin)
- Activity status

---

📌 Frontend:

- Dropdown filters (select inputs)

---

📌 Backend:
